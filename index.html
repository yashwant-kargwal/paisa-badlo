Paisa Badlo - Documentation

About Me

Hi, I'm Yashwant Kargwal, a web developer passionate about building useful web applications. Paisa Badlo is a simple and efficient currency converter that provides real-time exchange rates.

Project Overview

Paisa Badlo fetches real-time currency data using the following API:

https://cdn.jsdelivr.net/npm/@fawazahmed0/currency-api@latest/v1/currencies

This project is lightweight, user-friendly, and perfect for quick currency conversions.

How to Use

  1. Enter the amount you want to convert.
  2. Select the currency to convert from and to.
  3. Click the "Calculate" button to get real-time exchange rates.

API Usage

To fetch currency data, use the following JavaScript code:

fetch("https://cdn.jsdelivr.net/npm/@fawazahmed0/currency-api@latest/v1/currencies/${fromCurr.value.toLowerCase()}.json")
